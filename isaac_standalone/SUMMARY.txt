ğŸ’¡ KEY CONCEPTS
================================================================================

1. SimulationApp must be created BEFORE importing Isaac Sim modules
2. Use World() to create the simulation environment
3. Articulation represents robots with joints
4. Use ArticulationAction to control joint positions/velocities
5. Always call world.reset() before simulation
6. Use world.step(render=True) in the main loop

Solution: Normal! Isaac Sim modules only work with Isaac Sim Python

Solution: Check path in scripts matches your installation

Issue: Robot doesn't move
Solution: Use --mode animated or --mode interactive

Issue: Performance issues
Solution: Use --headless flag
